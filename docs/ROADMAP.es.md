# Hoja de Ruta hacia la Soberanía

Esta hoja de ruta describe el camino evolutivo desde un repositorio open-source hasta un estado digital soberano reconocido.

## Fase 1: El Kernel Soberano (Actual)

**Meta:** Definir especificaciones y construir la infraestructura del "Estado Mínimo Viable" (MVS).

- [ ] **Definición:** Manifiesto, Constitución v0, Modelo de Gobernanza.
- [ ] **Arquitectura:** Especificaciones para módulos de Identidad, Registro y Votación.
- [ ] **Comunidad:** Reunir desarrolladores núcleo y contribuyentes legales.
- [ ] **Mockups:** Diseño de interfaz del "Panel de Control del Estado".

## Fase 2: La Fundación Digital

**Meta:** Gobernanza operativa y existencia legal.

- [ ] **Envoltura Legal:** Registrar una DAO LLC o Fundación (ej. Wyoming, Islas Marshall, Suiza) para interactuar con el viejo mundo.
- [ ] **Ciudadanía Génesis:** Emitir primeros DIDs y credenciales de Ciudadanía (intransferibles).
- [ ] **Tesorería V1:** Tesorería multi-firma para financiar desarrollo.
- [ ] **Arbitraje Alfa:** Primera resolución de disputas usando Kleros o corte interna.

## Fase 3: Población y Economía

**Meta:** Crear valor interno y PIB.

- [ ] **Mercado de Servicios:** Ciudadanos ofreciendo servicios a ciudadanos (B2B, B2C).
- [ ] **Seguro Mutual:** Pools de seguro de salud/ingresos propiedad de los ciudadanos.
- [ ] **Registro de Negocios:** Compañías incorporadas estrictamente dentro de OpenSovereign.
- [ ] **Migración Masiva:** Onboarding de usuarios verificados de otras redes.

## Fase 4: Anclaje Territorial

**Meta:** Huella física.

- [ ] **Estrategia de "Embajadas":** Alquilar/comprar pequeños terrenos o hubs de co-working.
- [ ] **Alianza con Ciudades Charter:** Negociación con ZEDEs o Zonas Económicas Especiales para aplicar la ley OpenSovereign en una zona física.
- [ ] **Alcance Diplomático:** Primeros acuerdos bilaterales (aunque sean solo de comercio/reconocimiento) con naciones pequeñas.

## Fase 5: Soberanía

**Meta:** Reconocimiento Internacional.

- [ ] **Estatus de Observador:** Buscar estatus de observador en organizaciones internacionales.
- [ ] **Aceptación de Pasaporte:** Tratados de viaje bilaterales.
- [ ] **Reconocimiento Total:** El horizonte a largo plazo (cumplimiento Convención de Montevideo).
